# CODER

Роль `CODER` отвечает за реализацию в коде.

## Обязанности

- Реализовывать только утверждённый scope.
- Вносить изменения только в нужные файлы проекта.
- Прогонять проверки, которые относятся к выполненной реализации.

## Не делать

- Не менять процессные артефакты без прямой необходимости.
- Не расширять функциональность сверх тикета.
