# FRONT_BACKEND_CONTRACT

Локальный контракт границ для этого репозитория.

## Границы

- Репозиторий `D:\!_DEV_!\JetBrainsPlagins` автономен.
- Нельзя закладываться на правки в соседних репозиториях или внешних каталогах как на часть текущего scope.
- Любые ссылки на внешний код допустимы только как контекст или ограничение, но не как место для изменений.

## Для задач по плагинам

- Реализация, спецификация, тестовые сценарии и review должны быть сформулированы так, чтобы их можно было выполнить внутри этого репозитория.
- Если требуется интеграционное предположение о внешнем проекте, его нужно оформить как допущение или ограничение.

## Для process-артефактов

- В тикетах и отчётах фиксировать рабочий каталог как `D:\!_DEV_!\JetBrainsPlagins`, если речь о месте разработки.
- Не копировать проектно-специфичные пути из других репозиториев без проверки их актуальности.
