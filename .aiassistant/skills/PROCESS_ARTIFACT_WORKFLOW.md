# PROCESS_ARTIFACT_WORKFLOW

## Назначение

Повторяемый recipe для создания и обновления process-артефактов в `.windsurf/sandbox/MAIL/**` и `.windsurf/sandbox/00_STATE.md`.

## Когда использовать

- при создании `IN`/`OUT` артефактов;
- при handoff между ролями;
- при синхронизации состояния в `00_STATE.md`.

## Минимальный workflow

1. Прочитать активный тикет и обязательные входные артефакты.
2. Создать/обновить process-файл в корректной роли и директории.
3. Проверить имя файла, ссылки и соответствие ожидаемому формату.
4. Обновить `00_STATE.md` только по фактическому состоянию.
5. Добавить созданные/обновлённые process-артефакты в git index.

## Ограничение

Этот skill не может переопределять `rules/specs` или active ticket.
