# IDE_LANGUAGE_DETECTION

## Назначение

Повторяемый recipe для задач определения языка/контекста файла в IDE.

## Базовые шаги

1. Определить источники language metadata в текущем контексте IDE.
2. Зафиксировать правила fallback при недоступности контекста.
3. Проверить поведение на типовых сценариях из активного тикета.
4. Отразить ограничения и edge-cases в отчёте.

## Ограничение

Skill не является нормативным слоем и не переопределяет active ticket и `rules/specs`.
