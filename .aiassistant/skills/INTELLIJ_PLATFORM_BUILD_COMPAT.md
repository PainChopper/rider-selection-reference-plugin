# INTELLIJ_PLATFORM_BUILD_COMPAT

## Назначение

Повторяемый recipe для проверки совместимости и сборки плагина на IntelliJ Platform.

## Базовые шаги

1. Проверить версии платформы и инструментария в build-конфигурации.
2. Запустить целевые сборочные и проверочные команды проекта.
3. Зафиксировать ошибки совместимости и предложить минимальные исправления в рамках тикета.

## Когда использовать

- тикеты по build/toolchain;
- тикеты по совместимости версий IDE/Platform.

## Ограничение

Skill задаёт только workflow исполнения и не переопределяет `rules/specs/active ticket`.
